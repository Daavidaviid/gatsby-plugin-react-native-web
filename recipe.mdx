# Setup Gatsby with React-Native-Web / Expo

---

Install npm packages

`expo-av` is optional, just for the example page

<NPMPackage name="expo" />
<NPMPackage name="expo-av" />
<NPMPackage name="react-native" />
<NPMPackage name="react-native-web" version="~0.11.7" />
<NPMPackage name="gatsby-plugin-react-native-web" />

---

Install the plugin in gatsby-config.js

<GatsbyPlugin name="gatsby-plugin-react-native-web" />

---

Add a test page

It will be available at http://localhost:8000/react-native-web
